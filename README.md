# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/pallomagualter/podcastr.git
```

### 💻 Run Web Project

```bash
# Install Dependencies
$ yarn install

# Run Aplication
$ yarn dev
```
Go to http://localhost:3000/ to see the result.
